# Disaster_Safety_Application

## 概要
* Android用の防災情報アプリです。

## その他
### 使用API
| 種類             | API名                                           |
| ---------------- | ---------------------------------------------- |
| 天気情報           | OpenWeatherAPI（https://openweathermap.org/api ）|
| 実際の観測した地震| P2P地震API（https://www.p2pquake.net/develop/json_api_v2/ ）|
| 津波情報 | P2P地震API（https://www.p2pquake.net/develop/json_api_v2/ ）|
| 緊急地震速報（予報） | ❌（※利用できるAPIが無いため）|
| 緊急地震速報（警報） | P2P地震API（https://www.p2pquake.net/develop/json_api_v2/ ）|
